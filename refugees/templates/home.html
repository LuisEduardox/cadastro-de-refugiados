Criptografia de Dados Sensíveis

Todos os dados pessoais dos refugiados — nome, endereço, idade, renda, religião, filiação política e outros — são armazenados com criptografia de campo individual usando AES-256 via encrypted_model_fields.

Controle de Acesso por Papel

O sistema diferencia administradores de usuários padrão. Administradores gerenciam todos os registros; usuários padrão acessam apenas o próprio cadastro. O soft delete garante que nenhum dado seja permanentemente apagado sem rastreabilidade.
